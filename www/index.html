Build your own URL

The placeholder.pics service works uses a simple URL format to define the size, colours, and message used for your image. The format that image URLs must follow is:

  1. The URL must start with /svg/
  2. URLs that specify only a single dimension will be square, for example a request to /svg/100 will return an svg that is 100 pixels wide and 100 pixels high
  3. Width and height is specified via WIDTHxHEIGHT
  4. Background color can be either a single 3 or 6 character hex color code, or a gradient by defining a start color and stop color with START-STOP
  5. Foreground or text and border color can be a 3 or 6 character hex color code. Background color must be defined in order to set a foreground color
  6. A description can be set as the last part of the URL. Descriptions should be short and take into account the size of the image being requested. Foreground and background color is required when using a description
